# 조직 소개

## 프로젝트 개요
이 조직은 **술안주 추천 AI 냉장고** 프로젝트를 위한 협업 공간입니다.

프로젝트의 목표는 **주류와 보유 식재료를 기반으로 맞춤형 안주 레시피를 추천하는 AI 냉장고 프로토타입**을 구현하는 것입니다. 이를 위해 식재료 및 주류 인식, 센서 기반 이벤트 처리, 재고 관리, DB 기반 후보 조회, RAG/LLM 기반 추천 생성, 디스플레이 UI를 하나의 파이프라인으로 통합합니다.

## 저장소 구성

### `fe`
디스플레이 UI와 사용자 상호작용을 담당합니다.
- 현재 재고 조회 화면
- 추천 결과 화면
- 추천 상세 화면
- 인식 보정 화면

### `be`
시스템 전반의 서버/오케스트레이션 로직을 담당합니다.
- 센서 이벤트 처리
- 재고 상태 관리
- 추천 요청 생성
- DB 및 캐시 조회
- RAG/LLM 연계

### `ai`
모델 학습, 추론 최적화, 임베디드 배포 관련 기능을 담당합니다.
- 식재료 인식 모델
- 주류 인식 모델
- 데이터셋 관리 및 학습 파이프라인
- ONNX/TensorRT 변환
- Jetson 배포용 추론 모듈
